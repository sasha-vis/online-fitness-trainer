# Переменные окружения

В этом документе описаны переменные окружения, которые используются в проекте `online-fitness-trainer`.

## Переменные для разработки и работы приложения

Пример файла `.env` для локальной разработки:
[.env.example](../.env.example)

### Переменные для базы данных PostgreSQL

| Переменная          | Описание                                    | Пример значения    |
| ------------------- | ------------------------------------------- | ------------------ |
| `POSTGRES_USER`     | Имя пользователя базы данных PostgreSQL.    | `trainer_user`     |
| `POSTGRES_PASSWORD` | Пароль пользователя базы данных PostgreSQL. | `trainer_password` |
| `POSTGRES_DB`       | Название базы данных PostgreSQL.            | `trainer_db`       |

### Переменные для бэкенда приложения

| Переменная     | Описание                                              | Пример значения                                                               |
| -------------- | ----------------------------------------------------- | ----------------------------------------------------------------------------- |
| `DATABASE_URL` | URL подключения к базе данных PostgreSQL.             | `postgresql://trainer_user:trainer_password@db:5432/trainer_db?schema=public` |
| `NODE_ENV`     | Режим работы приложения (development или production). | `production`                                                                  |

### Переменные для фронтенда приложения

| Переменная     | Описание                                       | Пример значения  |
| -------------- | ---------------------------------------------- | ---------------- |
| `VITE_API_URL` | URL API бэкенда для использования в фронтенде. | `http://be:8080` |

## Переменные для CI/CD

Задавать переменные нужно на странице https://github.com/sasha-vis/online-fitness-trainer/settings/variables/actions

| Переменная                     | Описание                                               | Пример значения        |
| ------------------------------ | ------------------------------------------------------ | ---------------------- |
| `NEED_RUN_ACTION_DOCKER_BUILD` | Флаг для выполнения шага сборки Docker-образа в CI/CD. | `true` или `false`     |
| `DOCKERHUB_USERNAME`           | Имя пользователя Docker Hub для пуша образа.           | `sasha-vis`            |
| `DOCKERHUB_TOKEN`              | Токен доступа Docker Hub для аутентификации.           | `your_dockerhub_token` |
