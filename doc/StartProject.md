# Запуск проекта

Платформа-посредник между фитнес-тренерами и клиентами, которая позволяет тренерам составлять персонализированные программы тренировок и планы питания, а также контролировать выполнение этих планов клиентами.

## Настройка окружения

1. Убедитесь, что у вас установлены следующие инструменты:
    - Node.js (рекомендуемая версия: 22.x)
    - npm
    - Git

2. Клонируйте репозиторий:

    ```bash
    git clone git@github.com:sasha-vis/online-fitness-trainer.git
    cd online-fitness-trainer
    ```

## Установка зависимостей

Для первичной установки пакетов используйте команду:

    ```bash
    npm run install:all
    ```

Альтернатива - установка по отдельности:

    ```bash
    # Корневые зависимости
    npm ci

    # Зависимости клиента
    cd client && npm ci

    # Зависимости сервера
    cd server && npm ci
    ```

Эта команда установит все зависимости,
указанные в `package-lock.json`, и обеспечит чистую установку.

## Запуск в режиме разработки

Для запуска проекта в режиме разработки используйте следующие команды из скриптов в `./package.json`:

1. Запуск всего проекта (рекомендуется):

    ```bash
    npm run dev
    ```

2. Для фронтенда:

    ```bash
    npm run dev:client
    ```

3. Для бэкенда:

    ```bash
    npm run dev:server
    ```

4. Альтернативный запуск из подпапок

    ```bash
    # Фронтенд
    cd client && npm run dev

    # Бэкенд
    cd server && npm run dev
    ```

Теперь вы можете открыть браузер и перейти по адресу `http://localhost:3000`,
чтобы увидеть приложение в действии.
